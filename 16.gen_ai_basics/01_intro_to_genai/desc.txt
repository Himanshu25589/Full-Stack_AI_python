
# Introduction to Generative AI

## What is Artificial Intelligence (AI)?

Artificial Intelligence (AI) is a branch of computer science that focuses on creating systems capable of performing tasks that normally require human intelligence.

These tasks include:

* Understanding language
* Recognizing images
* Solving problems
* Learning from data
* Making decisions

### Examples

* ChatGPT
* Google Assistant
* Self-driving cars
* Face recognition
* Recommendation systems (Netflix, YouTube)

---

# What is Machine Learning (ML)?

Machine Learning is a subset of Artificial Intelligence.

Instead of programming every rule manually, a machine learns patterns from data and makes predictions or decisions.

### Simple Definition

> Machine Learning is the process of teaching computers to learn from data without being explicitly programmed.

### Example

A spam detection model learns from thousands of emails and predicts whether a new email is spam or not.

---

# What is Deep Learning (DL)?

Deep Learning is a subset of Machine Learning.

It uses artificial neural networks with multiple layers to learn complex patterns from large amounts of data.

Deep Learning is responsible for many recent AI breakthroughs.

### Applications

* Image Recognition
* Speech Recognition
* Language Translation
* Medical Diagnosis
* Self-driving Cars

---

# What is Generative AI?

Generative AI is a type of Artificial Intelligence that creates new content instead of only analyzing existing data.

It learns patterns from a large dataset and generates content that looks similar to the data it learned from.

### It can generate:

* Text
* Images
* Videos
* Audio
* Music
* Source Code
* 3D Models

### Example

Prompt:

```
Write a Python program to check whether a number is prime.
```

Output:

```python
num = int(input("Enter a number: "))

if num > 1:
    for i in range(2, num):
        if num % i == 0:
            print("Not Prime")
            break
    else:
        print("Prime")
else:
    print("Not Prime")
```
