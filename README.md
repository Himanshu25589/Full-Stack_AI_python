# Full-Stack AI Python

A beginner-friendly Python course covering everything from the basics to advanced topics used in AI and full-stack development.

---

## Who is this for?

This course is for **absolute beginners** — no prior programming experience needed. Each folder builds on the previous one, so just follow the numbered order.

---

## Course Structure

| Folder | Topic |
|--------|-------|
| `00_python` | Introduction to Python |
| `01_virtual` | Virtual Environments |
| `02_datatypes` | Data Types |
| `03_conditionals` | Conditionals (if/else) |
| `04_loops` | Loops (for/while) |
| `05_functions` | Functions |
| `06_chai_business` | Mini Project — Chai Business |
| `07_comprehensions` | List & Dict Comprehensions |
| `08_generators` | Generators |
| `09_decorators` | Decorators |
| `10_oop` | Object-Oriented Programming |
| `11_exceptions` | Exception Handling |
| `12_threads_concurrency` | Threads & Concurrency |
| `13_async_python` | Async Python |
| `14_pydantic` | Pydantic (Data Validation) |
| `challenges` | Practice Challenges |

---

## How to Use

1. Clone the repository
   ```bash
   git clone https://github.com/Himanshu25589/Full-Stack_AI_python.git
   ```

2. Navigate to any topic folder
   ```bash
   cd 00_python
   ```

3. Follow along with the files inside each folder in order.

---

## Requirements

- Python 3.10 or above
- A code editor (VS Code recommended)
