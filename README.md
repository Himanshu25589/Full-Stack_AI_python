A curated collection of notes, implementations, and practical projects documenting my progression from Python fundamentals to modern AI engineering.

The repository follows a structured learning roadmap covering Python, Generative AI, Retrieval-Augmented Generation (RAG), LangChain, backend engineering, and the technologies required to build scalable AI applications.

The concepts explored throughout this repository are continuously applied in production-oriented projects, including the **[Multi-Agent AI Research System](https://github.com/Himanshu25589/Multi-Agent-Research-System)** and the **[AI Video & Meeting Assistant](https://github.com/Himanshu25589/AI-Video-Assistant)**.

# Repository Structure

## Python Fundamentals

Building a strong programming foundation before moving into AI engineering.

| Folder | Covers |
|---------|--------|
| `00_python` | Python setup, syntax and variables |

| `03_conditionals` | Conditional statements |

## AI Engineering *(Currently Active)*

Understanding the core technologies behind modern LLM-powered applications.

| Folder | Covers |
|---------|--------|
| `16_gen_ai_basics` | Large Language Models (LLMs), Prompt Engineering, Transformer Architecture, Tokenization, Embeddings, OpenAI APIs and LangChain Fundamentals |
| `17_rag_pipeline` | Document Loading, Chunking Strategies, Embedding Models, Vector Databases, Similarity Search and Retrieval Pipelines |
| `18_langchain_runnables` | LangChain Expression Language (LCEL), Prompt Templates, Chat Models, Output Parsers, Structured Outputs and Workflow Composition |

Future modules will cover AI Agents, Model Context Protocol (MCP), Guardrails, Backend Engineering, Hybrid Search, Evaluation, and Deployment.

---

## Backend Engineering *(Planned)*

Building the backend infrastructure commonly used in production AI systems.

| Folder | Covers |
|---------|--------|
| `19_backend_node` | Node.js, Express.js, REST APIs, Routing and Middleware |
| `20_auth_database` | MongoDB, Authentication, JWT, Cookies and Sessions |
| `21_redis_docker` | Redis, BullMQ, Docker and Docker Compose |
| `22_postgres_pgvector` | PostgreSQL, pgvector and Semantic Search |
| `23_hybrid_search` | BM25, Reciprocal Rank Fusion (RRF) and Hybrid Search |
| `24_evaluation` | RAG Evaluation using Ragas |

The objective is to bridge backend engineering with modern LLM applications to build scalable, production-oriented AI systems.
